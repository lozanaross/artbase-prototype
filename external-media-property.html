External media

Property

Description

A qualitative expression of the degree of dependence on embedded media within an artwork, such as embedded links to sound, video, or still image files that are hosted outside the domain of the artwork website.

Associated values

Broken

Description: Missing external media has been observed by the auditing archivist, which causes significant damage to the artwork as a whole.

Damaged

Description: Some missing external media has been observed by the auditing archivist, but does not causes significant damage to the artwork as a whole.

Complete

Description: No significant missing external media has been observed by the auditing archivist in this artwork.

Cannot be determined

Description: During evaluation of this artwork, the auditing archivist was unable to determine the completeness of external media in this artwork.
