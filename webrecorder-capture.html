Webrecorder capture

Item
Artworks associated with this item

Description

This process involves the use of Rhizome’s web archiving software tool Webrecorder in order to create an archival WARC file. WARC files are re-performed via Rhizome’s Webenact platform. These re-performances aim to be as close to the original presentation of the artwork as possible, but still involve curatorial decisions taken by the archivist. See the Archival Plan metadata for more details.
